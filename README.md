## Configs:
- `API_HASH`
  - Get from [my.telegram.org](https://my.telegram.org) or [@UseTGzKBot](https://telegram.dog/UseTGzKbot).
- `API_ID`
  - Get from [my.telegram.org](https://my.telegram.org) or [@UseTGzKBot](https://telegram.dog/UseTGzKbot).
- `API_KEY`
  - Heroku API Key from [Here](https://dashboard.heroku.com/account).
- `APP_NAME`
  - Heroku App Name.
- `BOT_TOKEN`
  - Telegram Bot Token from [here](https://t.me/BotFather).
